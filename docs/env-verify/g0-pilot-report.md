# G0-pre 파일럿 리포트 — 근거충실 인용 학습→측정 루프

**일자**: 2026-07-17 · **장비**: RTX 5070 Ti 16GB Blackwell · **모델**: Qwen2.5-1.5B-Instruct (QLoRA)

## 실험

가설: **QLoRA 파인튜닝이 "근거(조문)가 주어졌을 때 「원문 인용」[조항ID] 형식으로 충실히 인용"하는 기술을 가르치는가?**

- 코퍼스: 헌법 제1~39조 93항목(closed-set)
- 학습 300건(train 조항) / 평가 18조항(**held-out — 학습 미포함**)
- 측정: `citation_verify`(결정적, LLM-judge 없음) — 인용 exact-match

## 해석 (정직)

**증명된 것:**
1. **학습→측정 전체 루프가 이 GPU에서 실동작** — base 로드 → 평가 → QLoRA → 재평가 단일 세션.
2. **파인튜닝이 근거충실 인용을 가르침** — base는 「」[id] 형식을 **전혀** 안 씀(0.0). 학습 후 held-out 조항에서 **verbatim 인용 + 정확 조항 지목 100%**.
3. **암기가 아닌 기술 전이** — 평가 조항은 학습에 없었고, 근거는 추론 시 제공(RAG식) → 모델은 "제공된 근거에서 인용하는 기술"을 학습해 미지 조항에 적용.
4. **VRAM 2.56GB** — 1.5B QLoRA는 극소. 7~14B로 스케일 여유 충분.

**증명되지 않은 것(과대해석 금지):**
- ❌ **"소형이 대형을 이긴다"(진짜 G0)** — 이 파일럿은 base(1.5B) vs 파인튜닝(1.5B) 자기비교. 소형 파인튜닝 vs 대형 base/GPT 교차비교가 아님. 진짜 G0는 별도.
- ⚠️ **단일 법령·형식 위주** — 헌법 하나. before 0.0의 상당 부분은 base가 정확한 「」[id] **형식**을 안 쓰는 것. 다양한 법령·모호질의·적대사례에서의 robust faithfulness는 미검증.
- ⚠️ **실세계 코퍼스 필요** — 진짜 신호는 국가법령정보 API 다법령 코퍼스(TA-08, user-gate 키 필요) + 검정력 산정 N + human anchor.

## 결론

**"측정을 소유하라" 테제의 학습 측면이 이 하드웨어에서 실제로 동작함을 실증.** 파이프라인(코퍼스→합성데이터→QLoRA→결정적 측정)이 엔드투엔드로 돌고, 파인튜닝이 근거충실 인용을 가르친다. 다음은 (1) 다법령 코퍼스(API 키) (2) 소형 vs 대형 교차비교 (3) K-FaithBench 정식화.
